### A1. AI 부트캠프 — 실제 적용 사례

부트캠프의 목적은 강의 청취가 아니라 **고객 팀이 직접 AI를 동료로 활용해 자기 업무를 자동화·개선할 수 있게 만드는 것**입니다.

- **바이브 코딩 워크숍 커리큘럼** — 시니어 개발자와 비개발 직군을 분리된 트랙으로 다루어, 개발자는 AI를 동료로 활용해 개발 속도를 높이고, 비개발 직군은 반복 업무를 직접 자동화해 운영 시간을 줄입니다.
- **Challenge Driven Learning** — HR 담당자·피플 매니저·교육생·운영자 네 역할을 명시적으로 분리한 교육 플랫폼. 현업 문제를 가지고 들어와 학습 성과가 조직 차원에서 측정·정착되도록 만들어, 교육 예산이 비용이 아닌 자산으로 남도록 설계했습니다.
- **글로벌 서비스 기업 한국 법인의 K-FSI 감사 대응 지원** — 정책·증적 정리·답변서 작성 워크플로를 AI와 함께 설계해 감사관 질의에 대한 응답 시간과 일관성을 끌어올리고, 내부 인력이 익명화·검증·답변 품질 관리를 직접 운영할 수 있는 가이드라인을 정착시켜 컴플라이언스 리스크를 구조적으로 낮췄습니다.

### A2. AI 파트너십 — 실제 적용 사례

파트너십의 목적은 **고객이 운전대를 잡은 채로 더 큰 변화를 빠르게 만들 수 있도록 옆에서 함께 가는 것**입니다.

- **국내 보안 SW 그룹사 바이브 코딩 도입 컨설팅** — 개발자 70명 대상 "바이브 코딩 현주소" 도입론 세션, 시니어 리더 20명 대상 모던 SW 엔지니어링 세션, 그리고 풀데이 도입 워크숍으로 이어지는 Why·What·How 3단계 프로그램을 설계·운영. 리더와 실무자가 같은 언어로 변화에 합의하도록 만들어, 조직 전체의 AI 도입 속도와 일관성을 함께 끌어올린 변화관리 프로젝트입니다.

### A3. AI 트랜스포메이션 자문 — 실제 적용 사례

자문의 목적은 **경영진이 방향을 결정할 때 필요한 정보와 실행 방법을 함께 제공하는 것**입니다. 그 근거를 로보코는 고객 PoC만이 아니라, AI를 연구원으로 활용한 자체 실험에서도 직접 만들어냅니다.

#### 고객 현장에서 검증한 도구
- **Persona Insight** — Git/GitHub·GitLab 활동 데이터를 Claude·GPT·Gemini 세 LLM으로 병렬 분석해 개발자·팀의 비즈니스 임팩트를 추론. Business 40 / Customer 25 / Code Quality 20 / Engineering Practice 15의 4차원 스코어링과 평가 근거 전면 공개 원칙. 별도 데이터 사이언스 조직 없이도 데이터에 근거한 인사·우선순위 결정이 가능함을 보여줍니다.

#### AI를 연구원으로 활용한 자체 실험 — 기술의 민주화 증명
> 예전에는 전담 클라우드 팀, 데이터 사이언티스트, 큰 인프라 예산이 있어야 가능했던 실험을, AI를 연구원처럼 활용해 1인·짧은 기간·적은 비용으로 직접 돌리고 결과를 공개합니다. 같은 접근을 고객 조직에도 그대로 이식합니다.

- **S3 Deep Dive** — Amazon S3를 단순 스토리지가 아닌 Key-Value Store, Event Store, 내구성 RDBMS(Litestream+SQLite), 서버리스 RDBMS(Athena), 파일 I/O 대안으로 밀어붙이는 실험. CDK로 배포 가능한 동작 코드와 전용 서비스 대비 정직한 트레이드오프 벤치마크를 함께 공개해, 전담 클라우드 연구팀 없이도 고객 아키텍처 결정의 가이드레일을 짧은 시간에 만들 수 있음을 입증.
- **Serverless Autoresearch** — H100을 8시간 점유해야 했던 Karpathy의 autoresearch 야간 자동 모델 개선 실험을 SageMaker Spot에서 재현. **2.3× 빠르고 5–18× 저렴하게**, H100 Spot에서 upstream 보고치 0.998을 능가하는 0.9951 val_bpb를 **약 $3.5에 달성**, **48개 실험 전체를 $3.94**로 완료. 8장의 핸즈온 튜토리얼도 함께 공개해 같은 접근을 누구나 복제할 수 있게 만들었습니다.
- **Serverless OpenClaw** — OpenClaw를 AWS 서버리스(Lambda Container + ECS Fargate Spot)에서 on-demand로 운영. 개인용 기준 **월 $1–2**, 콜드 스타트 1.35초, 유휴 시 비용 0원. ALB 대신 API Gateway 사용으로 고정비 월 $18–25를 제거한 "LLM 에이전트 서비스도 저비용으로 운영 가능하다"는 증명.

> 부트캠프에서 확인한 학습 장벽, 파트너십에서 검증한 조직 도입 패턴, 자문 프로젝트와 자체 실험에서 얻은 기술 검증 결과는 다시 다음 고객의 진단과 실행 계획에 반영되어, 로보코의 컨설팅 속도와 정확도를 계속 높입니다.
